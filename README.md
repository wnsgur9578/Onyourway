# OnYourWay

> 카카오맵 API 기반 여행 일정 웹 서비스
AI를 통해 간편하게 여행지를 설정하고 카카오톡으로 자동 알림을 받아보세요.
---

## 📆 프로젝트 기간

**2025.05.01 ~ 2025.05.28 (4주)**

---

## 📔 프로젝트 개요

- 카카오 맵 API 기반 여행 일정 수립 웹 서비스
- OPEN AI Api를 활용하여 챗봇 서비스 제공
- 수립된 일정에 대해서 한줄 요약 및 참고 사항 제공
- 카카오 SNS를 활용하여 간편 로그인 제공
- 원하는 시간에 맞춰 나에게 카카오톡을 보내 알림을 제공 

---

## 👥 팀원 소개

### 👤 장준혁

- **담당**
    - 기능
      - AI 챗봇 및 AI 요약 기능
      - 나에게 카카오톡 알림 보내기 기능
      - 원하는 시간에 맞춰 알림 자동화 서비스
    - UI/UX
      - 일정 요약 및 내 일정 모아보기 화면 구성
      - 챗봇 구성
      - 화면 구성 FIGMA 작업
     

---

### 👤 전원균

- **담당**
    - 기능
      - 일정 수립 서비스
      - 수립된 일정 수정 및 삭제 / 정렬화
      - 카카오 소셜 로그인 기능
      - ERD 작성
    - UI/UX
      - 메인화면 구성
      - 일정 수립 화면 구성
      - 내 프로필 화면 구성
